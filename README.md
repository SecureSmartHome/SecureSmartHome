# SecureSmartHome
Implementation of a Secure Smart Home based on Odroids and Android

# Currently supported Hardware
 - ODROID-XU4 (with Android)
 - ODROID Webcam
 - ODROID Weather Station
 - Reed Sensors
 - Door-Bell Button
 - Door Buzzer
